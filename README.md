# Creator Growth Control Plane

A high-performance automation engine and observability dashboard for scaling content creation and marketing operations. It provides a centralized control plane to manage and monitor automated workflows, enabling data-driven optimization.

## Key Features
*   **Automation Engine:** Orchestrates and executes content creation and marketing workflows.
*   **Observability Dashboard:** Real-time monitoring and visualization of campaign performance metrics.
*   **Production-Ready:** Built for reliability and scalability in hyper-growth environments.

## Tech Stack
*   Backend: Node.js, Express
*   Database: PostgreSQL
*   Frontend: React, TypeScript
*   Infrastructure: Docker, AWS

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/creator-growth-control-plane.git`
2.  Install dependencies: `npm install`
3.  Set up environment variables in a `.env` file.
4.  Run the development server: `npm run dev`
